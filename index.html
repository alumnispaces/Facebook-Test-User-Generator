Facebook Test User Generator

Create a fake Facebook user with login credentials to use in testing your Facebook integration.

View project on GitHub

Make-a-User 3000.

Using Facebook's /accounts/test-users/ API method, you can create fake Facebook login credentials for testing your Facebook app or integration on your website. Read more about the API endpoint in the Facebook docs.

To use this generator, you'll need:

  • Facebook App ID
  • App Access Token
